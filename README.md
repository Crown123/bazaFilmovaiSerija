#Baza filmova i serija
<table>
<tr>
<td>
 Ova veb-aplikacija koristi TMDB-Api da prikaže najpopularnije filmove i serije, filmove koji se trnutno prikazuju u bioskopima
  i omogućava pretragu fimova i serija po nazivu.
</td>
</tr>
</table>
